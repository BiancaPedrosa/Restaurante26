# Restaurante26 🍽️

Uma aplicação web simples para um restaurante, desenvolvida em **Node.js** utilizando o framework **Express**. O projeto serve páginas HTML, arquivos estáticos (CSS/Imagens) e renderiza um cardápio dinâmico lendo dados de um arquivo JSON.

## 🚀 Funcionalidades

- **Página Inicial:** Apresentação do restaurante.
- **Cardápio Dinâmico:** Leitura de produtos a partir de um arquivo `cardapio.json` e renderização em uma tabela HTML estilizada.
- **Fale Conosco:** Formulário de contato para os clientes enviarem mensagens.
- **Reservas:** Formulário para reserva de mesas com data e hora.

## 🛠️ Tecnologias Utilizadas

- Node.js
- Express (Roteamento e Servidor Web)
- HTML5 & CSS3
- JavaScript (Vanilla)

## ⚙️ Como executar o projeto na sua máquina

### Pré-requisitos
Antes de começar, você precisará ter o Node.js instalado na sua máquina.

### Passo a Passo

1. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/restaurante26.git
   ```
2. Acesse a pasta do projeto no terminal/cmd:
   ```bash
   cd restaurante26
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor:
   ```bash
   node app.js
   ```
5. Acesse no seu navegador: `http://localhost:3000`
